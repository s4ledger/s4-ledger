# HORIZON — Terminology Reference

Definitions HORIZON uses consistently in briefs. These are the common
terms across the procurement pipeline.

| Term | Meaning |
|------|---------|
| PR # | Pipeline Record number — the primary identifier for a procurement line item HORIZON tracks. Format `PR-#####`. |
| Hull | The vessel a pipeline record is associated with (e.g., T-AO 209). |
| COAR | Class of Acquisition / Replenishment classification. Shown as a chip. |
| RIY | Risk-Index Yield — composite slip-risk score, 0–100. |
| Baseline | The originally-planned date for a phase transition. |
| Actual | The realized date a phase transition occurred (or is currently forecast). |
| Variance | Actual minus baseline, in days. Positive = late. |
| Phase | One of: Definition, Procurement, Shipbuilder, Review, Award. |
| Status | One of: On Track, At Risk, Overdue, Complete. |
| Slip | The act of a phase missing its baseline; quantified in days. |
| Next action | The single most useful next step recorded against a PR. |
| Recovery plan | A documented action set to bring an At Risk row back to On Track. |

## Tone & formatting rules

- Always cite the PR number when discussing a specific record.
- Show dates as `YYYY-MM-DD`.
- Show variance as a signed integer with the unit `d` (e.g., `+12d`).
- Never editorialize on contracting decisions; only describe state and
  surface risk. Recommendations must be framed as options for the PM.
