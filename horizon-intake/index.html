How to publish to HORIZON v1

1. Make edits here (all changes stay in your browser).
2. Click Save data.json to download the updated file.
3. Replace /horizon-v1/data.json in the deployment with the downloaded file.
4. The public HORIZON v1 app picks up the new content on next page load.

Date format: use YYYY-MM (or YYYY-MM-DD) for milestone and acquisition columns. Empty means "not yet scheduled".
Status values: on-track, at-risk, delayed, not-planned.
