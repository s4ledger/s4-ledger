# GDPR Compliance Notes

Solus Protocol is designed to support GDPR compliance for EU partners:
- No personal data is stored on-chain; only SHA-256 hashes are anchored.
- All PHI and personal data is encrypted and stored off-chain.
- Data subjects can request deletion of off-chain records.
- Partners must ensure local storage and processing comply with GDPR.

For GDPR inquiries, contact support@solusprotocol.com.
